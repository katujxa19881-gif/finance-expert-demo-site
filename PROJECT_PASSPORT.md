## 8) Регламент обновления визитки в будущем
1. Локально внести изменения в `index.html` (и `assets/` при необходимости).
2. Закоммитить и отправить в `main` репозитория.
3. На сервере обновить код:

```bash
cd /var/www/expert.fin-dash.ru
git fetch origin main
git checkout -f main
git reset --hard origin/main
chown -R www-data:www-data /var/www/expert.fin-dash.ru
```

4. Проверить nginx:

```bash
nginx -t && systemctl reload nginx
curl -I https://expert.fin-dash.ru
```
